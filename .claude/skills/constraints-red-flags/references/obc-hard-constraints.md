# Ontario Building Code — Hard Constraints for Residential Development

These are **OBC requirements**, not zoning requirements. They cannot be varied by the Committee of Adjustment. If a project cannot meet them, relief must be sought through:
- An **Alternative Solution** submitted to the Chief Building Official (CBO), providing engineering proof of equivalent performance, or
- Appeal to the **Building Code Commission (BCC)** if the CBO refuses the Alternative Solution

---

## 1. Limiting Distances and Building Separation

The OBC regulates building separation through **"limiting distances"** — the measured distance from a building's exposing face to a property line, street/lane centre line, or imaginary line between two buildings on the same lot. This distance determines the **maximum percentage of unprotected openings (e.g., windows) permitted** on that wall.

### The 10-Minute Rule
If a local fire department **cannot guarantee first vehicle arrival within 10 minutes on 90% of calls**, the actual limiting distance must be **halved** for OBC calculations (e.g., a 1.2m setback is treated as 0.6m), unless the building is sprinklered.

### Consequences of the Halved Distance (If Not Sprinklered)
- Required distance to lot line increases to **2.4 metres**
- Side wall openings **prohibited**
- Soffits not permitted within **0.9 metres**

### Close Proximity Thresholds for Laneway / Garden Suites

| Available Limiting Distance | OBC Consequence |
|---------------------------|-----------------|
| < 2.4 m | No unprotected openings permitted; wall requires **¾-hour fire-resistance rating** |
| < 1.2 m | Exterior face must be clad with **non-combustible materials** |

## 2. Fire Department Access — Laneway Suites and Garden Suites

Because rear-yard suites are located behind primary dwellings, the OBC mandates strict emergency access routes.

### Mandatory Access Path

| Requirement | Standard |
|-------------|----------|
| Path width | Minimum **0.9 metres** unobstructed |
| Path height | Minimum **2.1 metres** unobstructed |
| Maximum travel distance | **45 metres** from street to suite entrance |
| Fire hydrant proximity | Within **45 metres** of where the fire vehicle parks on street |

Minor localized protrusions (hydro meters, gas meters) are permitted within the 0.9m width. If the property cannot provide this width, the owner must secure a registered **Limiting Distance Agreement** to share access width with a neighbouring property.

### Extended Travel Distance Option
Travel distance can increase to a **maximum of 90 metres** only if the designer incorporates **all** of:
- Automatic sprinkler system (Option 1), **OR** increased fire-protection building materials (Option 2)
- **Plus**: Exterior strobe light **and** interconnected smoke alarm/warning system **designed by a Professional Engineer**

### Cannot Be Varied by CoA
The 1.0m minimum unobstructed path and 45m maximum travel distance are **Ontario Building Code requirements**. The Committee of Adjustment has **no legal authority** to grant relief from these standards. A project that cannot achieve OBC fire access must pursue:
1. Alternative Solution to the CBO, or
2. A design change (relocate the structure to within travel distance)

## 3. Wildland-Urban Interface (WUI) Fire Constraints

The OBC **does not** currently have mandatory design standards for WUI construction. Historical attempts to incorporate them into the code have been rejected on the basis that local jurisdictions should determine requirements.

**Practical implications:**
- If a development is in a high-risk WUI area, the **municipality evaluates the risk** and may enforce mitigation measures
- Municipality can require non-combustible siding and roofing using:
  - **FireSmart Canada** principles (based on NFPA 1144)
  - **National Guide for Wildland-Urban Interface Fires**
- These are **locally imposed conditions**, not provincial OBC requirements — but they can be attached as conditions of Site Plan Approval or other discretionary approvals

## 4. Firefighting Water Supply Requirements

### Part 9 Buildings (Standard Houses — ≤ 600 m², ≤ 3 storeys)
- The OBC **does not** prescribe specific municipal water supply flows for standard residential homes
- The code assumes the arriving fire apparatus carries adequate water for initial search, rescue, and limited fire control
- Municipalities rely on **Fire Underwriters Survey (FUS)** calculations to mandate hydrant spacing and flow rates during the subdivision approval phase

### Part 3 Buildings (Large Residential — > 600 m² building area OR > 3 storeys)
- **Hard constraint**: The designer must explicitly confirm **adequate water supplies for firefighting** for each structure
- This is a mandatory OBC design exercise, not a guideline

## 5. What the Committee of Adjustment Cannot Vary

The CoA has **no legal authority** to vary OBC requirements. It operates strictly under the **Planning Act** and can only grant minor variances to Zoning By-law standards.

| OBC Requirement | CoA Can Vary? |
|-----------------|--------------|
| Minimum fire access path width (0.9m) | ❌ No — OBC |
| Maximum fire access travel distance (45m) | ❌ No — OBC |
| Limiting distance and unprotected opening limits | ❌ No — OBC |
| Fire-resistance ratings between units | ❌ No — OBC |
| Interconnected smoke/CO alarms | ❌ No — OBC |
| Secondary suite ceiling heights (1,850mm on stairs) | ❌ No — OBC |
| Lot coverage | ✅ Yes — Zoning By-law |
| Building height | ✅ Yes — Zoning By-law |
| Setbacks | ✅ Yes — Zoning By-law |

**Pipeline flag**: If a proposed project's physical layout results in a fire access path shorter than 0.9m or travel distance exceeding 45m (without sprinklers + PE design), this is a **blocking constraint**, not a variance candidate. Flag as `severity: "blocking"` in the constraints packet.

## 6. Key Definitions

| Term | Definition |
|------|-----------|
| **Limiting Distance** | Distance from an exposing building face to the property line, street/lane centreline, or imaginary line between buildings on the same lot |
| **Unprotected Opening** | Window, door, or other opening that is not fire-rated |
| **Exposing Building Face** | The exterior wall of a building that faces the limiting distance measurement |
| **Part 9 Building** | Residential building ≤ 3 storeys in building height and ≤ 600 m² building area |
| **Part 3 Building** | Building that exceeds Part 9 thresholds — requires more rigorous fire protection analysis |
| **Alternative Solution** | A proposed design solution submitted to the CBO that demonstrates equivalent performance to OBC acceptable solutions |
| **Building Code Commission (BCC)** | Provincial adjudicative tribunal resolving disputes on technical OBC requirements; the appeal body if CBO refuses an Alternative Solution |
